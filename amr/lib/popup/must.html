must (modal)   (AMR annotation guidance)

The modal must is generally annotated as obligate-01, with a :domain complement:

Example:  The boy must go.
     (o / obligate-01
           :ARG2 (g / go-02
                 :ARG0 (b / boy)))

Example:  The boy must not go.
     (o / obligate-01
           :ARG2 (g / go-02 :polarity -
                 :ARG0 (b / boy)))

Example:  The boy has to go.
     (o / obligate-01
           :ARG2 (g / go-02
                 :ARG0 (b / boy)))

In some cases, however, it conveys an inference ("epistemic" must):

Example:  John isn't here, he must be sick.
     (i / infer-01
           :ARG1 (s / sick-05
                 :ARG1 (p / person :wiki - :name (n / name :op1 "John")))
           :ARG2 (b / be-located-at-91 :polarity -
                 :ARG1 p
                 :ARG2 (h / here)))
